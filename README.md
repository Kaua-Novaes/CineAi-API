# 🎬 CineAi-API

Uma API RESTful inteligente para descoberta e recomendação de filmes usando IA generativa com Groq e LangChain.

## 📋 Sobre o Projeto

CineAi-API é uma aplicação Python que oferece endpoints inteligentes para explorar e obter recomendações personalizadas de filmes. A API integra modelos de IA de ponta através do Groq para fornecer recomendações contextuais e análises de filmes.

## 🎯 Funcionalidades

- 🤖 **Recomendações com IA** - Obtenha sugestões inteligentes de filmes usando LangChain e Groq
- 🎞️ **Banco de Dados de Filmes** - Acesso a um extenso catálogo de filmes
- 📚 **Análise de Conteúdo** - Processamento inteligente de informações sobre filmes
- 🔄 **CORS Habilitado** - Pronto para integração com frontends
- 📖 **Documentação Swagger** - Interface interativa para testar endpoints
- 🚀 **Deploy Ready** - Configuração para Render.com incluída

## 🛠️ Tecnologias Utilizadas

### Backend
- **Flask** - Framework web leve e poderoso
- **Flask-CORS** - Suporte para requisições cross-origin
- **Flask-Swagger-UI** - Documentação interativa da API

### IA & Machine Learning
- **LangChain** - Framework para aplicações com IA generativa
- **Groq** - API de modelos de linguagem de alto desempenho
- **Scikit-learn** - Ferramentas de machine learning
- **Pandas & NumPy** - Processamento e manipulação de dados

### Banco de Dados & ORM
- **SQLAlchemy** - ORM Python
- **SQLite** - Banco de dados embarcado (filmes.db)

### Deployment
- **Gunicorn** - Servidor WSGI para produção
- **Render** - Plataforma de deploy
